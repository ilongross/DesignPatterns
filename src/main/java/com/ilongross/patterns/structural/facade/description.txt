
Facade - позволяет объединить и скрыть в одном методе одного класса сложную реализацию
работы нескольких методов разных классов.


Применяется для скрытия реализации и для контроля выполнения работы.

Реализация:

1. Общий интерфейс
2. Общий суперкласс реализующий общий интерфейс.
3. В общем суперклассе с помощью композиции находятся экзмепляры остальных классов,
участвующих в процессе, но реализацию работы которых необходимо скрыть.
