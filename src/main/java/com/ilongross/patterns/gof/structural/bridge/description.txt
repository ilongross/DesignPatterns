
Bridge - применяется для создания взаимодействия двух иерархических сложных структур,
которые необходимо развивать независимо, но также необходимо поддерживать
взаимную работу этих структур.

Это структурный паттерн проектирования, который разделяет один или несколько классов на две отдельные иерархии - абстракцию и реализацию,
позволяя изменять их независимо друг от друга.
